## üë©‚Äçüè´ How to use this repository

Each day contains either contains a full D3 application or code samples that build up to the full app, following the same procedure used to create the article series. Most days can be run using the same basic steps (and for the days that require a bit more work, check out the tutorial series on the blog).

The steps to run standalone D3 applications:

```bash
Open the `index.html` file in your favourite browser.
```

The steps to run Webpack based D3 applications:

```bash
# install the dependencies
yarn install

# serve the project
yarn serve
```
